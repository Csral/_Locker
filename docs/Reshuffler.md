# Reshuffle
Reshuffle allows the program to pair a group of bits called 'Chunks' into a single 'Shuffle block' and then shuffle the entries within. The way it is shuffled depends on the algorithm chosen and the seed provided.

The first category of distinction is **chunk-wise** in which the bits within a chunk are not shuffled and **bit-wise** in which the bits within chunks are not secured and are shuffled throughout the Shuffle Block.
